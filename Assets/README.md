Термины
1. RPC — это способ вызвать метод на других игроках через сеть.
То есть ты вызываешь метод в своём коде, а он выполняется на всех клиентах (или только на выбранных).
2. Авторитет — это тот, кто управляет объектом в сети
   (кто «решает», где он находится, сколько у него HP, кто умирает и т.д.).
   
👑 В Photon есть два типа авторитета:
Тип	Кто управляет	Пример
Owner-based (владелец)	Тот, кто создал объект	Игрок управляет своим героем
Server/Master-based (сервер)	Только MasterClient	Нейтралы, боссы, лут, мобы
